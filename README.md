# HRMS Pro - Frontend UI

HRMS Pro Frontend UI is a modern, responsive, and dynamic Human Resource Management System built using React and Vite.

It provides an intuitive interface for managing employees, attendance, payroll, performance tracking, and HR operations.

---

## 🚀 Technologies Used

- React 18
- Vite
- CSS3
- Framer Motion
- Lucide React
- React Router DOM
- Axios

---

## ✨ Features

- Interactive Dashboard with real-time statistics
- Employee Management System
- Attendance & Leave Tracking
- Performance Management
- Payroll & Asset Management
- Notifications & Task Management
- Responsive UI for desktop and tablet devices
- Smooth animations and modern user experience

---

## 📦 Project Structure

```text
src/components/   → Reusable UI components
src/api/          → API service calls & Axios config
src/pages/        → Application pages
src/App.jsx       → Main application routing
public/           → Static assets
```

---

## ⚙️ Setup & Installation

### Prerequisites

- Node.js v18 or later
- npm or yarn

---

## 📥 Installation

Clone the repository and install dependencies:

```bash
npm install
```

---

## 🔧 Configuration

Update the backend API URL inside your API configuration file.

Example:

```javascript
baseURL: 'http://localhost:5000/api'
```

---

## ▶️ Run the Application

```bash
npm run dev
```

Application URL:

```text
http://localhost:5173
```

---

## 🔗 Backend Dependency

This frontend application requires the HRMS Pro Backend API to be running.
